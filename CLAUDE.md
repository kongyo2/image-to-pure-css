特殊用途のPure CSSのサブセット。通常のCSS知識がそのまま使えるが、以下の制約と追加機能がある。

---

## 禁止事項（これだけ避ければ他は普通のCSSと同じ）

| 禁止 | 理由 |
|------|------|
| `@keyframes` / `animation` | 無効。 |
| `url()` 関数（全箇所） | CSS全体が読み込まれなくなる |
| CSSコメント `/* */` | CSS全体が壊れる場合がある |
| `<style>` タグ | 無効。 |
| `<a href="...">` | 無効。 |
| JavaScript / `onclick` 等 | 全て無効 |
